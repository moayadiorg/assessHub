Expected CSV Format

# Required columns:
assessment_type,category,category_order,question,question_order,level_1,level_2,level_3,level_4,level_5

# Example row:
Cloud Maturity,Infrastructure,1,How automated is your infrastructure?,1,Manual,Some scripts,IaC templates,Full IaC with CI/CD,Self-service platform
Columns explained:
  • assessment_type - Name of the assessment (creates new if doesn't exist)
  • category - Category name within the assessment
  • category_order - Display order of the category (1, 2, 3...)
  • question - The question text
  • question_order - Display order within category
  • level_1 to level_5 - Descriptions for each maturity level
